## Metro Ranger Survival (SSWES Submission)
A Terminal-Style Survival Game developed in Python.

## Project Overview
This project was developed as a project to be submitted for the SSWES scheme. It is a survival resource manager. The player must navigate a dangerous tunnel system, managing health, ammunition, and air filters to reach a safe station.

## Features
- Resource Management: Real-time tracking of survival essentials.
- Randomized Encounters: Movement triggers and random events (mutant attacks, loot finds).
- Survival Mechanics: Suffocation automatically triggers when air filters are run out.
- Session limit : Tracks number of patrols and enforces a reset limit .

## How to Run
1. Ensure you have Python 3.10+ installed.
2. Clone this repository or download the source code.
3. Run the main script
